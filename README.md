Django Blog App

Features
* Adding Comment
* Sharing via email
* Suggest blogs with tags
* Search
* Sitemap
* RSS

Requirements to run:

Create the .env file in the main folder and fill it with the following variables.

EMAIL_HOST = ""
EMAIL_PORT =
EMAIL_HOST_USER = ""
EMAIL_HOST_PASSWORD = ""
DEFAULT_FROM_EMAIL = ""

DB_NAME=""
DB_USER='""
DB_PASSWORD=""
DB_HOST=localhost

Note: PostgreSQL was used as the database.
